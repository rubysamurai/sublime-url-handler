# Sublime Text URL handler for Linux

Open `subl://` URL scheme with Sublime Text 3.

## Installation

```
$ git clone git@github.com:rubysamurai/sublime-url-handler.git
$ cd sublime-url-handler
$ bash install
```

## Removal

```
$ cd sublime-url-handler
$ bash uninstall
```
